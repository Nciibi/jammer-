# IQ Samples Directory

This directory contains pre-generated IQ waveform files for HackRF jamming operations.

## Files

- `wifi_noise_20mhz.iq` - 20MHz QAM64 noise waveform (channel hopping)
- `wifi_noise_60mhz.iq` - 60MHz QAM64 noise waveform (full-band)

## Generation

These files are automatically generated by the HackRF jammer scripts when needed:

```bash
# Generate 20MHz waveform
siggen -f 2437000000 -r 20M -n 60 -m qam64 -o wifi_noise_20mhz.iq

# Generate 60MHz waveform  
siggen -f 2437000000 -r 60M -n 60 -m qam64 -o wifi_noise_60mhz.iq
```

## Specifications

- **Modulation**: QAM64
- **Center Frequency**: 2.437 GHz (WiFi Channel 6)
- **Duration**: 60 seconds
- **Power**: Normalized to unit variance
- **Format**: Complex IQ samples

## Usage

The jammer scripts will automatically generate missing files, but you can pre-generate them for faster startup.
